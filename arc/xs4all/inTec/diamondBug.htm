Diamond symbol is missing in some MS fonts

This page demonstrates the diamond symbol (HTML Entity &diam;) is missing from some fonts. I took the most used fonts, plus the generic CSS fonts. Other HTML entities (w3c, MS) can be looked at the extended page (see link below).

Under Windows XP, with Character Map application, the fonts Arial, Times New Roman and Courier do include the diamond character, under Unicode 2666. Verdana and Comic Sans don't have the character.
