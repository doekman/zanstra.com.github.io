Zanstra NET

Dit is een test om dynamisch style-sheets (CSS) te laden. Je kunt nu kiezen uit twee. Standaard wordt de eerste geladen. Als je een andere kiest wordt dit in een koekje opgeslagen, zodat je de volgende keer niet opnieuw hoeft te laden.

Het werkt als volgt: je zet het volgende in de <head> sectie van je html: 

<link rel="stylesheet" type="text/css" href="default.css" id="mainSheet">
<script type="text/javascript" src="cookie.js">
</script>

Dit zorgt er voor dat je een default style sheet hebt, en dat deze via de id-tag bereikbaar is via scripting. De script-regel zorgt ervoor dat we gemakkelijk met koekjes om kunnen gaan (je hebt dus ook het bestand cookie.js nodig).
