Dit is mijn persoonlijke archief van de nieuwsbrieven van het NRC Handelsblad - Webeditie. Toen ik op een gegeven moment terugkwam van vakantie, wilde ik weten wat er de afgelopen week gebeurd was. Ik kon echter geen overzicht vinden.

Toen kwam ik erachter dat ze ook een nieuwsbrief versturen. Daar ik kalenders leuk vind, ben ik maar wat met html, javascript en stylesheets aan de slag gegaan. Zie hier het resultaat.

Het kalender object is enigszins generiek en mag vrij gebruikt worden. Wellicht wordt dit object nog een keer geïntegreerd met mijn dateLib en komt dit in de dzLib rubriek van deze pagina's te staan (hoop doet leven). Tot die tijd moet je jezelf maar redden....

Heb je problemen om een HTML emailtje op te slaan als HTML bestand, of wil je plaatjes of geluiden uit een emailtje opslaan, terwijl ze niet als bijlage (attachment) voorkomen?

Daarvoor heb ik deze HTML Email tool/cursus-combinatie geschreven. Ben je niet technisch ? loop hard weg : klik hier ;

Daarnaast heb ik een stukje javascript geschreven, die de bijtjes (een zin geestig probeert op een nieuwsitem probeert in te spelen) met behulp van reguliere expressies uit de html bestanden haalt en in een XML bestand stopt. Er zit nu een style-sheet bij, die deze lijst groepeert per jaar per maand .

Hmmm, IE5 kan niet veel met m'n stylesheet. IE6 wel. Dus heb ik met de MSXML command-line tool html-versie gemaakt. Wel in UTF-16 (unicode) encoding. Ik heb nog geen optie gevonden om utf-8 te genereren.
